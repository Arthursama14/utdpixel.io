About UTD Pixel

UTD Pixel is a student project showcasing visualizations, maps, and interactive pixel art graphics for the UTD community.

This website uses a pixelated shooting star animation, where small white squares fall diagonally and twinkle like retro game stars against a gray-to-black night sky.
